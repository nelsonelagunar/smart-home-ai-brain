# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial project structure
- Device Manager with BroadLink discovery
- Pattern Learner with ML for automation
- LLM Client for natural language commands
- FastAPI REST API
- Docker and docker-compose support
- GitHub Actions CI/CD
- Comprehensive test suite
- OUI database integration for device identification

### Security
- Local-only processing (no cloud dependencies)
- No API keys required for basic functionality

## [0.1.0] - 2026-03-19

### Added
- First release
- Basic BroadLink IR/RF control
- Device discovery on local network
- Temperature/humidity sensor support
- Basic LLM integration via Ollama
- REST API endpoints
