# Roadmap

## 🎯 Visión

**Smart Home AI Brain** busca ser el cerebro inteligente de tu hogar, permitiendo:

1. **Descubrimiento automático** de dispositivos
2. **Control por voz natural** en español/inglés
3. **Aprendizaje de rutinas** para automatización
4. **Privacidad total** - 100% local
